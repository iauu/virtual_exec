# virtual_exec_parser

A parser crate to provide the syntax parsing, made available by `virtual_exec_parser::parse` 

**Note: This shouldn't be use directly(although possible). Use `virtual_exec_macro::parse` if 
build time optimization is necessary, or use `virtual_exec::exec` as the high-level API for 
the execution**
